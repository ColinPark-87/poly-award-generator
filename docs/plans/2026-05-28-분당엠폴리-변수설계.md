# 분당엠폴리 상장 변수 배치 Implementation Plan

> **For agentic workers:** 이 계획은 task별로 구현한다. 각 task는 렌더링→육안검증 사이클을 포함한다.

**Goal:** 분당엠폴리 두 상장(Grammar Certification / Certificate of Achievement)에서 변수(학생이름·반코드·월)를 템플릿 placeholder 위에 정확히 배치하고, 검은 밑줄을 지워 "가상의 선" 상태로 생성한다.

**Architecture:** 유성 캠퍼스의 `_scan_yuseong_placeholders`/`_render_yuseong` 패턴을 그대로 차용해 `_scan_bundang_placeholders`/`_render_bundang`를 신설하고 `build_certificate`에 `is_bundang` 분기를 추가한다. placeholder(`_____`)와 가로선을 PyMuPDF로 자동 감지 → 배경색으로 덮어 지움 → 폰트 자동맞춤 후 그 위치에 텍스트를 그린다.

**Tech Stack:** PyMuPDF(fitz) 좌표 감지, PIL 렌더링, NanumGothic(OFL) 한글 폰트.

## 변수 식별 (모든 변동 부분)

| 상장 | 변수 | 위치(@200dpi px) | 형식 | 폰트 |
|------|------|------------------|------|------|
| Grammar | 반코드 | `_____`(1637–1891, y1536–1647) 위쪽 줄 | `5HO1_1` (작게) | NanumGothic Regular |
| Grammar | 학생이름 | 같은 placeholder 줄 (크게) | `오민준 (Minjun Oh)` 한글+영문 | NanumGothic Bold |
| Grammar | 월 | "During the month of `______` 2026" (1830–1928, y2108–2148) | `April` | NanumGothic Regular |
| Grammar | (보류) | "on the `_____` POLY..." (1552–1634, y2049) | 미정 — 사용자 확인 필요 | — |
| LevelTop | 이름줄 | 굵은 가로선(y=887, x534–1495) 위 | `5HO2_2 Grace Jo` 반코드+영문 1줄 | NanumGothic Bold |
| LevelTop | 월 | 제목 "MPOLY `______`"(1339–1637, y349–449) | `April` | NanumGothic Bold |

**핵심 차이**
- Grammar 이름 = 2줄(반코드+한글영문), 한글 포함 → 한글 폰트 필수
- LevelTop 이름 = 1줄(반코드+영문), 한글 없음, 제목에 월 인라인

**미해결(사용자 확인)**: Grammar 본문 "on the ___ POLY Grammar Proficiency Test" 의 blank 값(연도? 월?). 기본은 연도("2026")로 두고 확인받는다.

## Task 3: `_scan_bundang_placeholders` — 좌표 감지

**Files:** Modify `generator.py` (유성 scan 함수 아래에 추가)

placeholder(`_____` span)와 가로선을 감지해 award_type별로 name/month/line 좌표를 반환한다.

- [ ] **Step 1:** 함수 작성

```python
@functools.lru_cache(maxsize=None)
def _scan_bundang_placeholders(pdf_path: str, dpi: int, award_type: str) -> dict:
    """
    분당엠폴리 템플릿 placeholder 감지.
    grammar: {'name': bbox, 'month': bbox, 'month_span': bbox}
    certificate_of_achievement: {'name_line': (y,x0,x1), 'month': bbox, 'month_span': bbox}
    모두 @dpi px.
    """
    sc = dpi / 72.0
    doc = fitz.open(pdf_path)
    page = doc[0]
    page_h = page.rect.height
    res = {"name": None, "name_line": None, "month": None, "month_span": None}

    spans = [s for b in page.get_text("dict")["blocks"] if b["type"] == 0
             for l in b["lines"] for s in l["spans"]]

    if award_type == "grammar_certification":
        # 이름: 순수 언더스코어 span 중 페이지 상단 75% 내 (가장 위 = 이름줄)
        name_cands = [s for s in spans
                      if s["text"].strip() and s["text"].strip().replace("_", "") == ""
                      and (s["bbox"][1] + s["bbox"][3]) / 2 < page_h * 0.75]
        if name_cands:
            s = min(name_cands, key=lambda s: s["bbox"][1])
            res["name"] = tuple(v * sc for v in s["bbox"])
        # 월: "month of" 포함 span 안의 언더스코어 위치
        for s in spans:
            if "month of" in s["text"].lower() and "_" in s["text"]:
                res["month_span"] = tuple(v * sc for v in s["bbox"])
                # 언더스코어 hit 중 이 span y와 일치하는 것
                for h in page.search_for("______") + page.search_for("_____") + page.search_for("____"):
                    if abs(h.y0 - s["bbox"][1]) < 4:
                        res["month"] = (h.x0 * sc, h.y0 * sc, h.x1 * sc, h.y1 * sc)
                        break
                break
    else:  # certificate_of_achievement
        # 이름줄: 굵은 가로선 (y 700~1100, 가장 긴 선)
        lines = []
        for dr in page.get_drawings():
            for it in dr.get("items", []):
                if it[0] == "l":
                    a, b2 = it[1], it[2]
                    if abs(a.y - b2.y) < 1.0:
                        lines.append((a.y * sc, min(a.x, b2.x) * sc, max(a.x, b2.x) * sc))
        mid = [l for l in lines if 700 * sc / (dpi/72.0) < 0]  # placeholder
        cand = [l for l in lines if 700 < l[0]/sc*72/72 or True]
        name_lines = [l for l in lines if (700*sc) < l[0] < (1100*sc)]
        if name_lines:
            res["name_line"] = max(name_lines, key=lambda l: l[2] - l[1])
        # 월: 제목 span "POLY ______" 안의 언더스코어
        for s in spans:
            if "_" in s["text"] and "POLY" in s["text"] and s["bbox"][1] < page_h * 0.4:
                res["month_span"] = tuple(v * sc for v in s["bbox"])
                for h in page.search_for("______") + page.search_for("_____") + page.search_for("____"):
                    if abs(h.y0 - s["bbox"][1]) < 6:
                        res["month"] = (h.x0 * sc, h.y0 * sc, h.x1 * sc, h.y1 * sc)
                        break
                break
    doc.close()
    return res
```

> 참고: 위 `mid`/`cand` 두 줄은 잘못된 잔재 — Step 2에서 제거(아래 정정본 사용).

## Task 4: `_render_bundang` — 텍스트 배치 + 밑줄 제거

**Files:** Modify `generator.py`

유성의 `_erase`(밝은 픽셀 평균 배경 샘플링) 헬퍼를 재사용해 placeholder/선을 지운 뒤 텍스트를 중앙 배치.

- [ ] **Step 1:** 함수 작성

```python
def _render_bundang(img, draw, template_path, award_type,
                    english_name, student_class, month, dpi=200):
    """분당엠폴리: 반코드·이름·월 배치 + 밑줄/선 제거(가상의 선)."""
    sc = dpi / 72.0
    ph = _scan_bundang_placeholders(template_path, dpi, award_type)
    month_name = month.rsplit(" ", 1)[0]  # "April 2026" → "April"

    kr      = config.BUNDANG_KR_FONT
    kr_bold = config.BUNDANG_KR_FONT_BOLD
    BLACK   = (0, 0, 0)

    def _bg_fill(x0, y0, x1, y1):
        """bbox 안 밝은 픽셀 평균 = 배경색."""
        samp = []
        sx_step = max(1, int((x1 - x0) / 30))
        for sx in range(int(x0), int(x1), sx_step):
            for sy in range(max(0, int(y0) - 30), int(y0)):
                try:
                    px = img.getpixel((sx, sy)); c = tuple(px[:3])
                    if sum(c) > 600: samp.append(c)
                except Exception: pass
        return (tuple(int(sum(c[i] for c in samp)/len(samp)) for i in range(3))
                if samp else (255, 255, 255))

    def _erase(x0, y0, x1, y1, pad=8):
        fill = _bg_fill(x0, y0, x1, y1)
        draw.rectangle([int(x0) - pad, int(y0) - 4, int(x1) + pad, int(y1) + 6], fill=fill)

    def _fit(font_file, text, max_w, max_size, min_size=24):
        size = max_size
        while size >= min_size:
            f = _load_font(font_file, size)
            tb = draw.textbbox((0, 0), text, font=f)
            if (tb[2] - tb[0]) <= max_w:
                return f
            size -= 2
        return _load_font(font_file, min_size)

    if award_type == "grammar_certification":
        nx0, ny0, nx1, ny1 = ph["name"]
        # placeholder 밑줄 지우기 (좌우로 넉넉히)
        _erase(nx0 - 200, ny0, nx1 + 200, ny1, pad=20)
        baseline_y = ny1  # 언더스코어 하단 = 이름 하단 기준선
        # 이름(크게) — 페이지 가로 중앙, max_w = 페이지폭 80%
        name_text = english_name
        max_w = int(img.width * 0.80)
        nf = _fit(kr_bold, name_text, max_w, 150)
        ntb = draw.textbbox((0, 0), name_text, font=nf)
        nx = (img.width - ntb[0] - ntb[2]) // 2
        ny = int(baseline_y) - ntb[3] - 10
        # 반코드(작게) — 이름 위
        if student_class:
            cf = _fit(kr, student_class, max_w, 56)
            ctb = draw.textbbox((0, 0), student_class, font=cf)
            cx = (img.width - ctb[0] - ctb[2]) // 2
            cy = ny - (ctb[3] - ctb[1]) - 24
            draw.text((cx, cy), student_class, font=cf, fill=BLACK)
        draw.text((nx, ny), name_text, font=nf, fill=BLACK)
        # 월
        if ph["month"]:
            mx0, my0, mx1, my1 = ph["month"]
            span = ph["month_span"]
            ex1 = span[2] if span else mx1
            _erase(mx0 - 6, my0, ex1, my1, pad=6)
            mf = _fit(kr, month_name, int((ex1 - mx0) * 1.6), int((my1 - my0) * 0.9))
            mtb = draw.textbbox((0, 0), month_name, font=mf)
            my = int(my0) + (int(my1 - my0) - (mtb[3] - mtb[1])) // 2 - mtb[1]
            draw.text((int(mx0), my), month_name, font=mf, fill=BLACK)

    else:  # certificate_of_achievement
        ly, lx0, lx1 = ph["name_line"]
        # 굵은 선 지우기 (가상의 선)
        fill = _bg_fill(lx0, ly - 20, lx1, ly - 5)
        draw.rectangle([int(lx0) - 10, int(ly) - 6, int(lx1) + 10, int(ly) + 8], fill=fill)
        # 이름줄: "반코드 영문이름" 1줄, 선 위 중앙
        name_text = f"{student_class} {english_name}".strip()
        max_w = int((lx1 - lx0) * 1.05)
        nf = _fit(kr_bold, name_text, max_w, 64)
        ntb = draw.textbbox((0, 0), name_text, font=nf)
        nx = int(lx0) + ((int(lx1 - lx0)) - (ntb[2] - ntb[0])) // 2
        ny = int(ly) - ntb[3] - 12
        draw.text((nx, ny), name_text, font=nf, fill=BLACK)
        # 월 (제목 인라인)
        if ph["month"]:
            mx0, my0, mx1, my1 = ph["month"]
            span = ph["month_span"]; ex1 = span[2] if span else mx1
            _erase(mx0 - 4, my0, ex1 + 4, my1, pad=4)
            mf = _fit(kr_bold, month_name, int((ex1 - mx0) * 2.2), int((my1 - my0) * 0.95))
            mtb = draw.textbbox((0, 0), month_name, font=mf)
            my = int(my0) + (int(my1 - my0) - (mtb[3] - mtb[1])) // 2 - mtb[1]
            draw.text((int(mx0), my), month_name, font=mf, fill=BLACK)
```

ABC = "abcdefghijklmnopqrstuvwxyz ABCDEFGHIJKLMNOPQRSTUVWXYZ"

          "July","August","September","October","November","December"]

       'certificate_of_achievement':'templates/분당엠폴리/certificate_of_achievement.pdf'}

    generator.build_certificate(at, name, cls, f"{month} 2026", out,
        template_override=TPL[at], campus='분당엠폴리')
    doc = fitz.open(out); pix = doc[0].get_pixmap(dpi=120)
    doc.close(); return pix

def test_name_alphabet_fits():

        _render(at, ABC, '5HO1_1', 'April', f'test_output/stress_{at}_abc.pdf')

        _render('grammar_certification', '오민준 (Minjun Oh)', '5HO1_1', m,
                f'test_output/month_grammar_{m}.pdf')
