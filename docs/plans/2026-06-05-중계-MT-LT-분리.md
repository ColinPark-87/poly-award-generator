# 중계 캠퍼스 MT/LT 업로드 분리 + LT 상장 문구 교체

작성일: 2026-06-05 / 대상: `상장생성기` (Streamlit)

## 배경 / 문제
- 중계는 그동안 성적 엑셀 1칸(MT/LT 혼합, 여러 파일)으로 받아 `select_winners`로 PS/HR/BW를 뽑았다.
- **버그**: 중계 상장 템플릿 3종(perfect_score/honor_roll/best_writer) PDF에 `on the Monthly Test` 문구가 텍스트(PlayfairDisplay-Italic 14pt)로 박혀 있어, **LT(레벨테스트) 성적으로 생성해도 "Monthly Test"** 로 나온다.
- 유성은 빈칸에 동적으로 "Monthly test."/"Level test."를 채워 이미 해결돼 있음(`generator.py:_LEVEL_TEST_MONTHS`).

## 확인된 사실
- 선정 로직(과목구성 차이)은 **이미 자동 대응**: `load_rows_from_excel`이 `Average`/`Lang. Composition` 등을 **헤더 이름으로** 찾으므로 MT/LT 열 위치·과목이 달라도 정상.
- 따라서 실제 수정 대상은 **(1) 업로드 UI 분리, (2) LT 상장 문구 교체** 두 가지.

## 설계 (중계 전용, 다른 캠퍼스 영향 0)

### 1. 업로드 UI (app.py) — `campus == "중계"`일 때만
- 성적 엑셀칸을 **MT 결과 엑셀(1개) / LT 결과 엑셀(1개)** 2칸으로 분리. Best SR CSV칸 유지.
- 월 감지: MT/LT 각 파일명에서 개별 추출(`mt_month`, `lt_month`), 못 찾으면 직접 입력.

### 2. 선정·생성 (app.py)
- MT 파일 → `select_winners` → 문구 "Monthly", 폴더 `MT/…`
- LT 파일 → `select_winners` → 문구 "Level",   폴더 `LT/…`
- 동일 `select_winners`·동일 기준 사용. 각 winner dict에 `_src`("MT"/"LT")·`_test_label` 태깅.
- 결과 표시: ps/hr/bw/sr 리스트에 MT/LT 합치고, 중계면 표에 **"시험"(MT/LT) 열** 추가. 미리보기 매칭은 `english_name+class+_src`로 구분.
- 둘 중 하나만 올려도 동작(한 시험만 처리).

### 3. 상장 문구 교체 (generator.py)
- `build_certificate(..., test_label: str = "Monthly")` 인자 추가.
- 중계(=기본 템플릿, non-yuseong/non-jungbal) 경로에서 `test_label=="Level"`이면 템플릿 PDF의 "Monthly Test"→"Level Test"로 치환 후 래스터라이즈.
- 신규 헬퍼 `_apply_test_label(doc, new_word)`: `_inject_director_signature`와 동일 패턴(스팬 탐지→배경색 덮기→TextWriter 재기입). 배경색은 **문구 오른쪽 빈 구간의 최대밝기 픽셀**로 샘플(크림 `255,254,249`). 폰트 `PlayfairDisplay-Italic.ttf`(신규, OFL) 14pt.
- 원장 사인 주입과 체이닝: doc 1개 열어 (사인)→(문구) 순으로 in-place 적용 후 1회 래스터.

### 4. 폰트 / 기타
- `fonts/PlayfairDisplay-Italic.ttf` 추가(Google Fonts OFL, weight 400 italic). config에 `BODY_ITALIC_FONT` 상수.
- 중계 경로용 모듈 reload 훅 추가(분당과 동일 패턴, 세션 1회) — 핫리로드 시 신규 generator 반영.

## 검증
- 단위: `_apply_test_label` 후 텍스트에 "Level Test" 존재 & "Monthly Test" 부재, 폰트 로드.
- 통합: 중계 LT로 build_certificate → 생성 PDF 텍스트에 "Level Test", MT는 "Monthly Test" 유지.
- 시각: 치환부 PNG 크롭 육안 확인(배경 박스 없음, italic 일치).
- 기존 테스트 스위트 전체 통과(다른 캠퍼스 회귀 없음).
